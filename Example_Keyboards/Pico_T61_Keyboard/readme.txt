Thinkpad T61 USB or Bluetooth Keyboard Controller using a Raspberry Pi Pico W 

*****Beware, this project is still in development*************************. 
The keyboard was tested with KMK and all keys are working (see KMK folder). 
The TrackPoint was tested with Arduino C code and is also working (see Arduino folder).
Code to combine the keyboard and TrackPoint has not yet been developed.
